图书背后的故事

2019-12-18

图书背后的故事

我所说的图书背后的故事，真的是字面意思。随便找一本你手头能够拿到的书，忽略封面华丽的装饰（可能并没有），或者是书脊上厚重庄严的烫金标题，当然也可能是——

不过这些都不是今天的重点。我们把书翻到背面，你看见了什么？

对，这个条形码就是今天的主角

一些闲话

人类总是很热衷于将文字记录在某个物体上，古代的美索不达米亚人把楔形文字记录在软泥版上，中国人把甲骨文刻在乌龟壳上，古埃及人会把文字写在莎草纸上，不一而足。这些行为最初的目的是服务于自己的，比如今天隔壁家的张三欠了我五条鱼，那我得把事情记下来怕忘了。渐渐的就发展成为了宗教行为，比如甲骨文就是祭司才掌握的文字，文字和书写被附会上了神灵的力量。

从中国人发明印刷术起（最早的还是雕版印刷），书写就不再是一件私密的事情了，一份文字可以被高效率地重复千百万遍，被更多的人阅读，这就是图书。在古代，国际交流尚不发达的时候，只有很少的人能掌握大量书籍，每册书的印刷数量也不够多，除了一些非常重要的典籍，大部分的书籍也还是手抄本，也可能是全世界只有一册的孤本。作为一名计算机专业的学生，这种非常不利于保护数据安全的情况不禁让我一身冷汗。

在历史上就出现过这样的悲剧：公元前三世纪由托勒密一世兴建的亚历山大图书馆(Royal Library of Alexandria)是同时代世界上最大的图书馆，据说在其鼎盛时期有大约400000册书籍。可惜在公元三世纪左右，亚历山大图书馆殁于战火，其中的藏书尽失。

这也说明了古代保存书籍是非常困难的一件事，任何天灾人祸都有可能让一段知识在历史长河中消失。

到了现代，我们已经有太多的出版物了，技术的发展让很多垃圾一般的文字也能有机会被印刷出来。我们要担心的反而不是能不能很好的保存书籍，而是如何去管理这些浩如烟海的书籍。

正题

现在书店里售卖的书籍背后都会有一个条形码，一般来说，这个条形码的下方和上方分别会有一串13位的数字（对有些较早出版的书，上面的数字是10位的）。其实，条形码的信息表示的也就是这13位的数字，只是换了一种可能让机器识别的方式罢了。

比如在本文开头展示的条形码上方数字是978-7-121-31383-7，这串数码的名称叫“国际标准图书码”(ISBN, International Standard Book Number)，可以看到，它被-隔成了五段，下面分别来介绍一下这五段的意思

  1. 978：任何一本图书前面的三位都是978或者979，表示图书前缀
  2. 7：语言或地区编码，7说明这是一本由中国大陆出版的书
  3. 121：出版机构编码，121说明这是一本由电子工业出版社出版的书
  4. 31383：出版编码，这串数字是由出版社出版书籍的时候给出的编码，确定唯一的一本图书
  5. 7：校验码，这个内容放到后面去讲

搜索一下就知道，这本书是由电子工业出版社出版的《数字图像处理》，学计算机视觉的学生都绕不开的坎。

仔细一些的人可能会发现，在图书馆里收藏的一些旧书的扉页里也会有ISBN码，不过有些可能是10位的。这就要从ISBN的历史说起了

早在1965年，都柏林圣三一学院的统计学教授Gordon Foster和Emeritus就提出了一种9位的书籍标记系统，用于对图书做唯一的标识，称为标准图书码（SBN, Standard Book Number）。之后在1967年，由英国的David Whitaker提出了国际标准图书码的构想，另外在1968年的时候，美国的Emery Koltay也参与了之后ISBN码的发展。到了1970年，国际标准化组织（ISO）把ISBN作为一种国际标准加入到了ISO 2108中。四年之后，发明SBN的英国也加入到了ISBN的怀抱中。

在1970年施行的这种ISBN码是10位的，和13位的ISBN相比，它少了前缀的三位数字，而且最末一位的校验码部分也不同。13位的ISBN码是在2007年施行的，所以目前出版的新书背后基本都是13位。

目前来说，所有前缀是978的13位ISBN码都可以被顺利转换到10位，比如前面的这本《数字图像处理》的10位ISBN码就是7-121-31383-9，至于最后这一位是怎么计算的，有兴趣的同学请移步到下一节。

ISBN码非常方便对图书进行登记，因为按“地区-出版社-出版顺序”这样的方式指定的书籍是不会重复的，即使是一本小说分了上下两册，或者一套百科全书分了70多本，都不会妨碍每本书的记录。

相比来说，国内图书馆按照“中国图书馆分类法”或者美国图书馆使用的“杜威十进制分类法”是按照图书的内容分类的，这样会在每一个子条目下得到非常多的图书，也有可能一本图书因为跨学科的原因，有几种分类的方法。所以在大学入学的时候，图书馆都要对学生做一些培训，不然借个书都借不到。

下面的内容比较硬核

不管是10位ISBN还是13位的ISBN，最末一位都是一个校验码，它可以用来检验一个ISBN码是不是正确的，对于10位ISBN码，具体的计算方法如下：

将ISBN的数码从左到右依次记做x1x_1x1​，x2x_2x2​，x3x_3x3​，x4x_4x4​，x5x_5x5​，x6x_6x6​，x7x_7x7​，x8x_8x8​，x9x_9x9​，x10x_{10}x10​，那么把它们按下面的方法计算出来

10x1+9x2+8x3+7x4+6x5+5x6+4x7+3x8+2x9+x10 10x_1 + 9x_2 + 8x_3 + 7x_4 + 6x_5 + 5x_6 + 4x_7 + 3x_8 + 2x_9 + x_{10} 10x1​+9x2​+8x3​+7x4​+6x5​+5x6​+4x7​+3x8​+2x9​+x10​

把这个和除以111111，得到的余数如果是000，那么就是正确的ISBN码，否则就是错误的。比较数学的说法是，对于一个正确的ISBN码，它总满足

(10x1+9x2+8x3+7x4+6x5+5x6+4x7+3x8+2x9+x10)≡0 mod 11 (10x_1 + 9x_2 + 8x_3 + 7x_4 + 6x_5 + 5x_6 + 4x_7 + 3x_8 + 2x_9 + x_{10})\equiv 0\ \mathrm{mod}\ 11 (10x1​+9x2​+8x3​+7x4​+6x5​+5x6​+4x7​+3x8​+2x9​+x10​)≡0 mod 11

反向来看，如果前999位的数字是正确的，那么如果得到校验码呢？具体的计算方法是这样的：

x10=(11−(10x1+9x2+8x3+7x4+6x5+5x6+4x7+3x8+2x9) mod 11) mod 11 x_{10} = (11 - (10x_1 + 9x_2 + 8x_3 + 7x_4 + 6x_5 + 5x_6 + 4x_7 + 3x_8 + 2x_9)\ \mathrm{mod}\ 11)\ \mathrm{mod}\ 11 x10​=(11−(10x1​+9x2​+8x3​+7x4​+6x5​+5x6​+4x7​+3x8​+2x9​) mod 11) mod 11

这里的x mod 11x\ \mathrm{mod}\ 11x mod 11就是xxx除以111111之后的余数。可以证明，这样计算得到的校验码是可以满足前面检测方法的。

另外，一个数除以11得到的余数有可能会是10，那么这个时候校验位就记为X，这也就是为什么有些10位的ISBN码最后一位会莫名其妙出现一个字母的X。

相比来说，13位的ISBN码在设计上就简明扼要得多了，由于前缀可以是979，所以记录书籍的数量比10位的ISBN整整多出了一倍，而校验位不再会出现X这样的字母了。对于一个合法的13位ISBN码，具体的检测方法如下：

将ISBN的数码从左到右依次记做x1x_1x1​，x2x_2x2​，x3x_3x3​，x4x_4x4​，x5x_5x5​，x6x_6x6​，x7x_7x7​，x8x_8x8​，x9x_9x9​，x10x_{10}x10​，x11x_{11}x11​，x12x_{12}x12​，x13x_{13}x13​，，那么它们服从

(x1+3x2+x3+3x4+x5+3x6+x7+3x8+x9+3x10+x11+3x12+x13)≡0 mod 10 (x_1 + 3x_2 + x_3 + 3x_4 + x_5 + 3x_6 + x_7 + 3x_8 + x_9 + 3x_{10} + x_{11} + 3x_{12} + x_{13})\equiv 0\ \mathrm{mod}\ 10 (x1​+3x2​+x3​+3x4​+x5​+3x6​+x7​+3x8​+x9​+3x10​+x11​+3x12​+x13​)≡0 mod 10

而校验位的计算则变成了

x13=10−(x1+3x2+x3+3x4+x5+3x6+x7+3x8+x9+3x10+x11+3x12) mod 10 x_{13} = 10 - (x_1 + 3x_2 + x_3 + 3x_4 + x_5 + 3x_6 + x_7 + 3x_8 + x_9 + 3x_{10} + x_{11} + 3x_{12})\ \mathrm{mod}\ 10 x13​=10−(x1​+3x2​+x3​+3x4​+x5​+3x6​+x7​+3x8​+x9​+3x10​+x11​+3x12​) mod 10

那么这个余数就只有0∼90\sim90∼9了

像ISBN这种在物理世界传播的数码，可能会被各种各样的因素干扰，比如条形码被墨水涂得看不清了，图书馆编目人员在手动输入的时候输错了等等。有一个校验位就可以保证我们随时能检测出来一个ISBN码是不是错的。

题外话

在计算机科学中，校验码和更强的纠错码是非常重要的发明，它们在信息的正确传输中起到了举足轻重的作用。它们有些存在于互联网，保证你的消息能正确地被女神收到；有些存在于你的手机和电脑，保证文件可以正确存储在硬盘里。

看吧，书都不用翻开，就有这么多东西要学，还是好好读书吧。
