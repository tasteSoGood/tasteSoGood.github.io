使用jq处理json文件

2019-11-05

Json是一种轻量级的标记语言，可以用接近人类可读的方式来记录和表示数据之间的关系，可以作为跨语言的数据交换文件

在Linux中有一个命令行工具jq，可以很方便地在命令行中处理Json格式的文件，提取其中的信息

安装

jq的官网地址在这里，在Linux的各大发行版的软件源中都有这个工具，但一般需要用户自行安装

安装完成以后，可以通过man jq来查看文档信息

使用

我们假设有一个文件为sample.json，命令行调用一般有两种写法

  • 直接调用

  • 配合cat使用管道调用

两种方法都是可以的，可以根据自己的喜好来选择

由于有管道的作用，我们以下都称Json对象

在Json对象大致可以分为两种：列表和字典，其中支持的数据类型（字符串和数字）就不是这里的内容了，我们寻找对象，一般以列表和字典为依据

列表结构的处理

  • jq '.'，得到全体

  • jq '.[1:4][]'，得到子列表的全体元素

  • jq '.[1,2,3]'，按下标选择列表元素

  • jq '.[-1]'，从后向前寻址

字典结构的处理

对于下面这样的字典结构

以下列出一些例子以及它们可以得到的效果

  • jq '.'

  • jq '.[]'

  • jq 'keys'

  • jq 'keys[]'，即列表操作，取出所有列表元素

  • jq '.<key-1>'，提取特定键的值

    注意，这里的键名在没有歧义的时候是可以不加引号的，但是如果键名含有空格，就一定要用双引号包裹

并列输出

并列输出使用的是,来分隔开多个部分，同样是上面字典的例子，如果用

  • jq '.<key-1>, .<key-2>, keys'，可以得到

只要是合法的列表和字典操作，都可以用,来并列输出
